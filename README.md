# Praktikum 3

Kode ini adalah program Python sederhana yang memeriksa apakah sebuah bilangan adalah kelipatan 3, lalu membandingkan dua bilangan lain untuk menemukan yang terbesar. Ini berdasarkan diagram alur yang diberikan dan cocok untuk latihan pemrograman dasar.

Alur Program

Mulai : Menampilkan judul program.

Input Bilangan Utama : Pengguna memasukkan bilangan bulat.

Cek Kelipatan 3 : Jika bilangan habis dibagi 3 (sisa = 0), hasil "BESAR"; jika tidak, "KECIL".

Input Bilangan Pembanding : Pengguna memasukkan dua bilangan (A dan B).

Bandingkan Terbesar : Jika A > B, terbesar = A; jika tidak, terbesar = B.

Output : Menampilkan hasil cek dan bilangan terbesar.

Selesai : Program berakhir.
